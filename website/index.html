How it works

ForeLLM gives you practical, hardware-specific model recommendations in three steps.

1) Detect hardware

Reads your available RAM, CPU cores, GPU backend, and VRAM to understand your real capacity.

2) Score models

Evaluates models across quality, speed, fit, and context so rankings are useful, not generic.

3) Run locally

Use the top picks in your local stack and launch through providers like Ollama, llama.cpp, or MLX.

Install in seconds

Pick your platform, copy the command, and start checking what runs on your machine. Scoop and Homebrew require the package to be in their registries; if a command fails, use Quick install or From source.

Windows (Scoop)

scoop install forellm

macOS / Linux (Homebrew)

brew install forellm

Quick install (Linux/macOS)

curl -fsSL https://raw.githubusercontent.com/emireln/forellm/main/install.sh | sh

From source (all platforms)

git clone https://github.com/emireln/forellm.git && cd forellm && cargo build --release

Requires Rust. Binary: target/release/forellm
