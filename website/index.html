Why ForeLLM

Hardware-aware choices instead of trial and error.

Hardware-Aware RAM, CPU, GPU, VRAM fit
Multi-Mode TUI, CLI, API, GUI
Runtimes Ollama, llama.cpp, MLX
Advanced Multi-GPU, MoE

How it works

Three steps from hardware to running models.

1) Detect hardware

Reads RAM, CPU cores, GPU backend, and VRAM so recommendations match your machine.

2) Score models

Ranks by quality, speed, fit, and context so the list is useful, not generic.

3) Run locally

Use top picks with Ollama, llama.cpp, or MLX—copy the command and go.

Features

What you get out of the box.

TUIInteractive terminal: navigate models, filter, and check fit in real time.
CLISystem info, fit ranking, search. Script and automate easily.
REST APIRun forellm serve for integrations and automation.
Desktop GUILauncher first: open the GUI, run Agent in terminal, or run ForeLLM TUI—one click, no commands. Model Explorer (copy, download, link to Hugging Face), theme toggle (dark/light/system). Home button returns to launcher.
Agent ForeAI chat (GUI or CLI): streaming, file attachments, image analysis (vision model), tools (read docs, analyze images, Python, web search, run commands). Export chat to Markdown or TXT. Multiple agents. Start from launcher “Run Agent in Terminal” or npm run agent in forellm-gui; /help for commands.
Multi-GPUCombined GPU memory awareness for multi-card setups.
MoECorrect memory and run-mode handling for Mixtral, DeepSeek-V2, etc.
ProvidersOllama, llama.cpp, MLX—detect installed models and run commands.
QuantizationPicks the best quant that fits your VRAM/RAM.

Install

Copy the command for your platform and run it.

Windows (Desktop GUI)

Launcher, Model Explorer, Agent Fore. Built with npm run dist in forellm-gui.

Download Setup.exe

Antivirus / SmartScreen: Windows may show a warning because the installer is not code-signed. This is common for open-source apps. Choose "More info" → "Run anyway" (or "Run anyway" in SmartScreen). The source code is public on GitHub; you can build it yourself with npm run dist.

Windows (Scoop)

scoop install forellm

macOS / Linux (Homebrew)

brew install forellm

Docker

docker build -t forellm . && docker run --rm -it forellm

From repo root. Run CLI: docker run --rm forellm fit --json

Quick install (Linux/macOS)

curl -fsSL https://raw.githubusercontent.com/emireln/forellm/main/install.sh | sh

From source

git clone https://github.com/emireln/forellm.git && cd forellm && cargo build --release

Requires Rust. Binary: target/release/forellm
